Authentication#

Overview#

Download a JSON keyfile and point to it using an environment variable:

$ export VOXJAR_APPLICATION_CREDENTIALS="/path/to/keyfile.json"

JSON Keyfile#

Example:

{
        "url": "https://api.voxjar.com:9000/",
        "token": "token"
}
